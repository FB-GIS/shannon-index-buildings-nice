##What is the Shannon Index?  

The Shannon Diversity Index, initially designed to analyze ecosystem biodiversity, is also highly relevant for urban studies:  

It quantifies the balance and richness of different building uses within a given area.  
A higher index value indicates greater functional diversity, reflecting a mix of uses (residential, commercial, cultural, etc.).  
A lower index value suggests homogeneity in building functions, often dominated by a single use (e.g., purely residential or industrial).
